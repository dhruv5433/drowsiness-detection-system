# Drowsiness Detection System

A real-time driver drowsiness detection system that monitors eye activity using a webcam and triggers an audio alert when drowsiness is detected — helping prevent road accidents.


## Features
- Real-time face and eye tracking via webcam
- Detects drowsiness using Eye Aspect Ratio (EAR)
- Instant voice alert using text-to-speech
- Lightweight and runs on standard hardware



## Tech Stack
- **Python**
- **OpenCV** – Video capture and image processing
- **MediaPipe** – Face mesh and landmark detection
- **pyttsx3** – Offline text-to-speech alerts



## Installation
```bash
pip install opencv-python mediapipe pyttsx3
```



##  How to Run
```bash
 drowsiness.py
```
Make sure your webcam is connected.
